# Control Flow

This folder contains examples of control flow structures in JavaScript: conditional statements, switch cases, and truthy/falsy values.

## Files in this folder:

- **one.js** - If/else statements, logical operators (&&, ||), conditional chaining
- **switch.js** - Switch statement syntax and use cases
- **truthy.js** - Truthy and falsy values in JavaScript

## Key Concepts Covered:

### Conditional Statements:
- `if` statement
- `if...else` statement
- `else if` chaining
- Ternary operator
- Logical operators:
  - `&&` (AND)
  - `||` (OR)
  - `!` (NOT)

### Switch Statement:
- Switch syntax
- Case matching
- Default case
- Break statements

### Truthy and Falsy Values:

**Falsy values** (evaluates to false):
- `false`
- `0`
- `-0`
- `0n` (BigInt zero)
- `""` (empty string)
- `null`
- `undefined`
- `NaN`

**Truthy values**: Everything else evaluates to `true`

## Best Practices:

- Use explicit comparisons when checking for specific values
- Be aware of truthy/falsy coercion in conditionals
- Use switch for multiple discrete values
- Use logical operators for concise conditionals
