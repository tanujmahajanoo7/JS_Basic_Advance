# JavaScript Basics - Part 3

This folder focuses on functions, scopes, and execution contexts in JavaScript.

## Files in this folder:

- **01_functions.js** - Function declarations, parameters, arguments, return statements
- **02_scopes.js** - Scope types (global, function, block), scope chain
- **03_arrow.js** - Arrow function syntax and differences from regular functions
- **04_iife.js** - Immediately Invoked Function Expressions (IIFE)

## Key Concepts Covered:

### Functions:
- Function declarations vs function expressions
- Parameters vs arguments
- Default parameters
- Rest parameters (`...args`)
- Returning values from functions
- Passing objects and arrays to functions

### Scopes:
- Global scope
- Function scope
- Block scope (introduced with `let` and `const`)
- Scope chain and lexical scoping

### Arrow Functions:
- Concise syntax: `() => {}`
- Implicit return
- `this` binding differences

### IIFE:
- Immediately invoked function expressions
- Use cases for IIFE

## Important Notes:

- Functions are first-class citizens in JavaScript
- Arrow functions don't have their own `this` binding
- Function scope vs block scope affects variable accessibility
- Rest parameters gather remaining arguments into an array
