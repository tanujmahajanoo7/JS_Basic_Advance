# JavaScript Basics - Part 1

This folder contains fundamental JavaScript concepts including variables, data types, type conversions, comparisons, strings, numbers, math operations, and date handling.

## Files in this folder:

- **01_variables.js** - Variable declarations (const, let, var), scope differences, and best practices
- **02_dataTypes.js** - JavaScript data types (primitive and reference types)
- **03_conversionOperation.js** - Type conversion and type coercion operations
- **04_comparision.js** - Comparison operators (==, ===, !=, !==, etc.)
- **05_strings.js** - String manipulation methods and operations
- **06_nums_and_math.js** - Number operations and Math object methods
- **07_datesinJs.js** - Date object creation, manipulation, and formatting
- **datatypes-summary.js** - Summary of JavaScript data types

## Key Concepts Covered:

- Variable declarations: `const`, `let`, `var`
- Block scope vs function scope
- Primitive types: number, string, boolean, null, undefined, symbol, bigint
- Type conversion and coercion
- Comparison operators (strict vs loose equality)
- String methods and template literals
- Math operations and properties
- Date object and date manipulation

## Best Practices:

- Prefer `const` for values that won't change
- Use `let` for variables that need reassignment
- Avoid `var` due to scope issues
- Use strict equality (`===`) instead of loose equality (`==`)
