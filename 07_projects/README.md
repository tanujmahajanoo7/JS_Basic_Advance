## Projects Overview:

### Project 1: Color Changer
Button-based background color changer using event listeners and DOM manipulation.

### Project 2: BMI Calculator
Form-based BMI calculator with input validation and result display.

### Project 3: Digital Clock
Real-time clock using `setInterval` and `Date` object.

### Project 4: Guess the Number Game
Interactive number guessing game with validation, game state management, and reset functionality.

### Project 5: Keyboard Event Listener
Displays keyboard event details (key, keyCode, code) in a table format.

### Project 6: Random Color Generator
Start/stop color changing background using `setInterval` and random color generation.

## Key Concepts Demonstrated:

- Event handling
- DOM manipulation
- Form validation
- Game state management
- Timer functions (`setInterval`, `clearInterval`)
- Event object properties
- Random value generation
