# JavaScript Basics - Part 2

This folder covers arrays and objects in JavaScript - two fundamental data structures.

## Files in this folder:

- **01_arrays.js** - Array creation, methods (push, pop, unshift, shift), includes, indexOf
- **02_array.js** - Additional array methods and operations
- **03_objects.js** - Object creation, properties, and basic operations
- **04_objects.js** - Advanced object operations and methods

## Key Concepts Covered:

### Arrays:
- Array declaration and initialization
- Array methods:
  - `push()` - Add element to end
  - `pop()` - Remove element from end
  - `unshift()` - Add element to beginning
  - `shift()` - Remove element from beginning
  - `includes()` - Check if array contains element
  - `indexOf()` - Find index of element
  - `slice()` - Extract portion without modifying original
  - `splice()` - Add/remove elements (modifies original)
  - `join()` - Convert array to string

### Objects:
- Object literal syntax
- Property access (dot notation vs bracket notation)
- Object methods and properties
- Object manipulation

## Important Notes:

- `slice()` does not modify the original array
- `splice()` modifies the original array
- Arrays in JavaScript are dynamic and can hold mixed types
